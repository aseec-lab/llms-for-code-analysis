The `DrawableUtils` class provides utility methods for working with drawables.

The class defines various constants and fields to support its operations.

The `canSafelyMutateDrawable()` method checks if a drawable can be safely mutated.

The `fixDrawable()` method fixes an issue with vector drawables on Android 5.0 (API level 21).

The `fixVectorDrawableTinting()` method fixes tinting issues with vector drawables.

The `getOpticalBounds()` method retrieves the optical bounds of a drawable.

The `parseTintMode()` method parses a tint mode integer and returns the corresponding `PorterDuff.Mode`.
