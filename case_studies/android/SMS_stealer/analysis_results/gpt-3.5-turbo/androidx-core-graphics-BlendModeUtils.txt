The `BlendModeUtils` class in the `androidx.core.graphics` package provides utility methods for working with BlendModeCompat and PorterDuff.Mode. 

The `obtainBlendModeFromCompat()` method maps a BlendModeCompat enum value to the corresponding BlendMode enum value. It returns the corresponding BlendMode, or null if no match is found.

The `obtainPorterDuffFromCompat()` method maps a BlendModeCompat enum value to the corresponding PorterDuff.Mode enum value. It returns the corresponding PorterDuff.Mode, or null if no match is found.
