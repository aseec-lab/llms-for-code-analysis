The `TintableImageSourceView` interface defines methods for getting and setting the tint color and mode for an image source. 

- The `getSupportImageTintList` method returns the color state list for the image tint.
- The `getSupportImageTintMode` method returns the PorterDuff mode for the image tint.
- The `setSupportImageTintList` method sets the color state list for the image tint.
- The `setSupportImageTintMode` method sets the PorterDuff mode for the image tint.
