The Helper interface is defined in the file Helper.java. It has three methods:

1. void add(ConstraintWidget paramConstraintWidget): This method is used to add a ConstraintWidget to the Helper.

2. void removeAllIds(): This method is used to remove all the ids associated with the Helper.

3. void updateConstraints(ConstraintWidgetContainer paramConstraintWidgetContainer): This method is used to update the constraints of the Helper with respect to the given ConstraintWidgetContainer.
