- The `R.dimen` class defines constant integer values for dimensions.
- The `R.drawable` class defines constant integer values for drawable resources.
- The `R.id` class defines constant integer values for view IDs.
- The `R.integer` class defines constant integer values.
- The `R.layout` class defines constant integer values for layout resources.
- The `R.string` class defines constant integer values for string resources.
- The `R.style` class defines constant integer values for styles.
