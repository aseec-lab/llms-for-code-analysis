The `ScaleGestureDetectorCompat` class in the `androidx.core.view` package provides compatibility methods for working with the `ScaleGestureDetector` class.
- The `isQuickScaleEnabled` method checks if quick scale is enabled on the given `ScaleGestureDetector` object. It uses the `isQuickScaleEnabled` method available from API level 19.
- The `setQuickScaleEnabled` method sets the quick scale enabled or disabled on the given `ScaleGestureDetector` object. It uses the `setQuickScaleEnabled` method available from API level 19.
